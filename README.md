# Karma Coin Authenticator
A Firebase authenticaiton proxy micro service

## Running in dev mode
```bash
dart lib/src/server.dart
```

## Building 
```bash
dart compile exe lib/src/server.dart -o ./authenticator.exe
```

## Running
```
./authenticator.exe
```
